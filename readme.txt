== Installation ==

To install our plugin simply upload its files to your WordPress plugins directory:

1. Download the plugin and upload the files to `/wp-content/plugins/sqoot/`
2. Activate the plugin through the Plugins menu in WordPress
3. Customize the widget from Settings > Sqoot Widget

Want to get paid, too?

1. Sign up at [Sqoot.com](http://www.sqoot.com/)
2. Copy your Affiliate Token and paste it into Settings > Sqoot Widget
3. Save your settings

== Frequently Asked Questions ==

= How do I make money? =

We share half of all commissions when deals are purchased through your blog.
We only make money when you do.

[Register](http://www.sqoot.com/#signin) to track everything via your dashboard.

= How do I make more money? =

The name of the game is "customize"! Once the plugin is installed be sure to customize
the deals for your readers with the categories on the settings page.

= Can I use my existing WordPress theme? =

Yup. It installs in 1 click.

= Does this work for WordPress.com? =

You can’t install plugins to WordPress.com hosted blogs but you can get our
[JavaScript snippet](http://www.sqoot.com/) yourself and install it as a
[WordPress widget](http://automattic.com/code/widgets/use/) or 
[add it to your blog's theme](http://codex.wordpress.org/Editing_Files#Using_the_Theme_Editor_and_Plugin_Editor).
